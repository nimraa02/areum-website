# AREUM

AREUM is a modern luxury handbag brand website inspired by minimalist aesthetics and timeless elegance. This project is designed to showcase a clean and sophisticated online shopping experience while strengthening my front-end web development skills.

## Features
- Modern and elegant landing page
- Responsive design
- Luxury-inspired typography and color palette
- Product collection showcase
- About section
- Smooth CSS animations

## Built With
- HTML5
- CSS3

## Project Status
This project is currently under development. Additional pages, interactive features, and JavaScript functionality will be added in future updates.
